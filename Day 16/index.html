Paragraphs: <p>

This is a simple paragraph. Paragraphs are block-level elements that automatically add margin above and below. They're perfect for body text in articles, blog posts, and general content.

This is another paragraph. Notice the space between paragraphs? That's the default margin. You can control this with CSS if needed.

Paragraphs can contain bold text, italic text, links, and even code snippets. They're versatile containers for text content.

Preformatted Text: <pre>

Without <pre> (Regular Paragraph):

function hello() { console.log("Hello World"); return true; }

With <pre> (Preserved Formatting):

function hello() {
    console.log("Hello World");
    return true;
}

Key difference: The <pre> tag preserves:

  • Spaces
  • Line breaks
  • Tabs
  • Text formatting exactly as typed

Comparison: <p> vs <pre>

Paragraph (<p>)
