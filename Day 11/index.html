Text Editing

  • Select All: Ctrl + A
  • Copy: Ctrl + C
  • Paste: Ctrl + V
  • Undo: Ctrl + Z
  • Save: Ctrl + S

Browser Shortcuts

  • New Tab: Ctrl + T
  • Refresh: F5 or Ctrl + R
  • Developer Tools: F12 or Ctrl + Shift + I

Remember these essential shortcuts:

  • To go to end of line: End
