üí™ Practice Exercise

Build a YouTube Video Portal

Requirements:

  1. Responsive Video Player:
    • Main player area that adapts to screen size
    • Custom controls using YouTube API
    • Video info display (title, description, views)
  2. Video Gallery:
    • Grid of 6+ videos with thumbnails
    • Search/filter functionality
    • Sort by date, views, or duration
    • Lazy loading for better performance
  3. Playlist System:
    • Create and manage multiple playlists
    • Add/remove videos from playlists
    • Playlist progress tracking
    • Auto-play next video
  4. User Features:
    • Watch later list
    • Video ratings (like/dislike)
    • Playback speed control
    • Subtitles language selection
  5. Technical Implementation:
    • Use YouTube IFrame API for all controls
    • Store user data in localStorage
    • Responsive design with CSS Grid/Flexbox
    • Accessibility features (keyboard nav, ARIA)
    • Error handling for failed loads

Starter Structure:

<div class="video-portal">
    <!-- Header -->
    <header>
        <h1>My Video Portal</h1>
        <div class="search-bar">
            <input type="text" placeholder="Search videos...">
        </div>
    </header>
    
    <!-- Main Content -->
    <main>
        <!-- Video Player -->
        <section class="player-section">
            <div id="mainPlayer"></div>
            <div class="player-controls">
                <!-- Custom controls -->
            </div>
        </section>
        
        <!-- Video Gallery -->
        <section class="gallery-section">
            <div class="video-grid" id="videoGrid">
                <!-- Videos loaded here -->
            </div>
        </section>
        
        <!-- Playlists -->
        <section class="playlist-section">
            <div class="playlists" id="playlists">
                <!-- User playlists -->
            </div>
        </section>
    </main>
</div>
