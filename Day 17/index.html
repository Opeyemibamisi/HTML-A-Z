<q> (Inline Quote)

  • Short quotations within text
  • Browser adds quotation marks automatically
  • Use cite attribute for source URL
  • Semantic markup for short quotes
  • Quotes can be styled with CSS quotes property

Example:

She said Hello World! and smiled.

<blockquote> (Block Quote)

  • Longer quotations (multiple paragraphs)
  • Styled as separate block with indentation
  • Often includes <footer> for attribution
  • Can contain <cite> for source title
  • Use cite attribute for source URL

Example:

This is a longer quote that deserves its own space.
